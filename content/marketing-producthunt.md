## Long Description

### Intro
```
Choosing an AI model shouldn't require a PhD in machine learning.

Yet every comparison you find is either:
❌ Written by the companies themselves (biased)
❌ Outdated (pre-2024 models)
❌ Missing the models you actually want to compare
❌ Focused on benchmarks that don't match your use case
```

### Was ist AgentRank?
```
AgentRank is an independent comparison platform for Large Language Models.

We analyze 20+ models across:
• Performance benchmarks (MMLU, HumanEval, Arena Elo)
• Real API pricing (per-token costs, no surprises)
• Context windows and limitations
• Best use cases for each model
• Open source vs commercial tradeoffs
```

### Key Features
```
🔍 Smart Search
Find models by capability, price, or provider. Filter by what matters to you.

⚖️ Side-by-Side Comparison
Compare up to 3 models with detailed spec sheets and benchmark scores.

📊 Real Benchmarks
No marketing numbers. Actual test results from MMLU, HumanEval, and LMSYS Arena.

💰 True Cost Analysis
API pricing + self-hosting costs. Calculate TCO for your volume.

🤖 Open Source Included
Llama, DeepSeek, Qwen - not just OpenAI and Anthropic.
```

### Use Cases
```
For Developers
Find the best coding model. Compare GPT-5, Claude 3.5, and open source alternatives.

For Startups
Optimize AI spend. Route queries to the right model and save 70%+ on API costs.

For Enterprises
Evaluate models for compliance, privacy, and performance requirements.

For Researchers
Access consolidated benchmark data and model specifications.
```
