## Technische SEO-Checkliste

### On-Page SEO
- [ ] Title-Tag: 50-60 Zeichen, Keyword vorne
- [ ] Meta-Description: 150-160 Zeichen, Call-to-Action
- [ ] H1: Einmal pro Seite, Haupt-Keyword
- [ ] H2-H6: Strukturierte Überschriften
- [ ] Bilder: Alt-Text mit Keyword
- [ ] URLs: Kurz, Keyword, Bindestriche
- [ ] Interne Links: 3-5 pro 1000 Wörter
- [ ] Externe Links: Zu autoritativen Quellen

### Content-Qualität
- [ ] Mindestens 1500 Wörter für Pillar Content
- [ ] Mindestens 800 Wörter für Cluster Content
- [ ] Aktuelle Daten (2025)
- [ ] Tabellen für Vergleiche
- [ ] Bullet-Points für Lesbarkeit
- [ ] Zusammenfassung am Ende

### Technisch
- [ ] Mobile-optimiert
- [ ] Ladezeit < 3 Sekunden
- [ ] HTTPS
- [ ] Schema.org Markup
- [ ] XML Sitemap
- [ ] Canonical Tags
